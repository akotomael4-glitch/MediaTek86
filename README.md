# MediaTek86

## Présentation

MediaTek86 est une application de gestion des absences dévellopée entièrement en C# avec le logiciel Visual Studio avec une base de données MySQL.

Cette application permet de :
- gérer les personnels
- gérer les services
- gérer les absences
- gérer les motifs d'absence
- sécuriser l'accès grâce à une authentification


## Technologies utilisées 

- C#
- Windows Forms
- MySql
- Visual Studio
- GitHub


## Fonctionnalités 

### Gestion du personnel

- affichage des personnels
- ajout d'un personnel
- modification d'un personnel
- supression d'un personnel


### Gestion des services

- affichage des services
- association d'un personnel à un service

### Gestion des motifs

- affichage des motifs d'absence

### Gestion des absences 

- affichage des absences
- ajout d'une absence
- modification d'une absence
- suppression d'une absence

### Sécurité 

- connexion avec login et mot de passe
- accès sécurisé grâce à la table responsable

### Vérification
- contrôle des dates
- détection des chevauchements d'absence
